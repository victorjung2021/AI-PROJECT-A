비지도학습 : 답을 알려주지 않고 학습

인공지능 데이터셋(dataset)
인공지능 개발을 위해 하습을 위한 관측데이터의 집합
데이터셋 = 학습데이터(이미지,글씨) + 정답(라벨)이 같이 있는 것

인공지능데이터는
훈련용, 평가용(테스트용) 으로 구분( 8:2 )

<인공지능 객체탐지 방식>
* 중요함.

디지털트랜스포메이션(DT,DX)
Digital Transformation
디지털화된 데이터와 기술을 통합, 활용하여
새로운 가치를 창출하는 혁신활동
=> 빅데이터활용, AI

메타버스(metaverse)
현실공간이 아닌 거대한 디지털 공간으로
가상세계와 현실세계를 연결하는 확장 개념

클라우드(cloud)
인터넷 상의 가상화된 서버에서 프로그램을 두고
필요할때마다 컴퓨터나 스마트폰으로
불러오는 서비스

빅데이터(Bigdata)
기존의 데이터베이스 관리도구의 능력을 넘어서
대량의 모든 데이터를 처리하는 방식

빅데이터 3V
속도(Velocity), 규모(Volume), 다양성(Variety)

빅데이터 6V
3V + 진실성(Veracity), 시각화(Visualization), 가치(Value)
=> 빅데이터의 6V에 가장 중요한것 = 진실성(Veracity)

빅데이터 데이터 구조
정형, 반정형, 비정형 <= 문제

YOLO란?
You Only Look Once의 약자로 한번 보이는것만으로도
Object의 종류와 위치를 추측하는 딥러닝 알고리즘
=> 데이터라벨링 형식 :  텍스트

Segmentation
이미지를 의미있는 단위로 분할하여 객체들을
분류하는 인공지능 알고리즘

얼굴인식(Face Landmark)
인공지능에서 사람의 얼굴의 정보를 알아내기
위하여 5개 또는 68개의 점으로 사람의 특징을
분류해내는 알고리즘
=> dlib, mediapipe

아나콘다(anaconda)
Python라이브러리들을 쉽게 설치하고
관리하는 도구

데이터라벨링(data labeling)
사랑이 만든 사진, 문서등의 데이터를 인공지능이
학습할 수 있는 형태로 가공하는것
=> 데이터라벨링형식
     PascalVOC(XML), YOLO(TEXT), CreateML(JSON)

전처리
인공지능이 학습하기 좋게 데이터를 고치는
모든 작업

신경망구조
입력층, 은익층, 출력층

순환신경망(RNN)
계층의 출력 순환 신경망
음성, 웹이브폼, 텍스트등

합성곱신경망(CNN)
합성곱레이어와 풀링레이어의 구조로 학습하는 알고리즘
채널 : 이미지데이이터를 처리하는 방식
         컬러 = 3채널, 흑백 = 1채널
특성지도 : 이미지의 특정부분을 Convolution
을 통해 줄이는 방법을 통해 이미지의 특징을
만드는 과정
패딩(Padding)
합성곱연산을 하기전 주변에 값을 덧붙여서 눌리는것

생성적 적대 신경망(GAN)
생성자 = Generator
구분자(판별자) = Discriminator
=> 데이터증대, 이미지 복원

StyleGAN
GAN의 변형형태로 이미지 생성시 스타일을 저정하는 장점
=> 진짜같은 가짜이미지를 무한대로 생성

CUDA
앤베디아(NVIDIA)가 개발한 소프트웨어로
인공지능 학습속도를 향상시키는 프로그램

티처블머신
구글에서 개발하여 코딩없이 머신러닝을
할 수 있는 프로그램

크롤링(crawling)
web상에서 존재하는 정보(Contents)를 수집하는 작업

BeautifulSoup
인터넷(HTML)정보를 원하는 데이터를 쉽게
나누어(parsing)해주는 함수
